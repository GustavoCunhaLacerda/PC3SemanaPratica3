## 💻 Como testar e utilizar

### Requisitos

**Clone o repositório e acesse a pasta**

```bash
$ git clone https://github.com/GustavoCunhaLacerda/PC3SemanaPratica3.git && cd PC3SemanaPratica3
```

**Siga esses passos**

```bash
# Install the dependencies
$ npm install

# Start the client
$ npm start
```
